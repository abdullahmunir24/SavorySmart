## Description
SavorySmart is a smart recipe recommendation and meal planning application designed to help users discover new recipes and plan their meals efficiently.

## Features
- Recipe discovery and recommendations
- Meal planning and scheduling
- Ingredient management
- User preferences and dietary restrictions

## Technologies Used
- React.js
- Tailwind CSS
- Node.js
- Express.js

## Getting Started

### Prerequisites
- Node.js installed on your machine
- npm or yarn package manager

### Installation
1. Clone the repository
```bash
git clone https://github.com/abdullahmunir24/SavorySmart.git
```

2. Navigate to the project directory
```bash
cd SavorySmart
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm start
```

## Usage
Once the application is running, you can:
- Browse through recipe recommendations
- Create custom meal plans
- Manage your ingredient inventory
- Set dietary preferences
